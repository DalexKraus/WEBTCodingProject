Project Charter

This is the project charter for our website, but mainly our web-app.

The Client

The client for this project ist David Klewein, our Teacher in Web Development.

The project managers

The project managers are David Kraus and Samuel Kowatschek. To find out more about us, visit the about page.
We make this project as a part of a school project in the subject Web Development.

The project

The project is about self-driving cars, especially about the different level of autonomous vehicles. The goals of the project is to learn more about self-driving cars and also learn game mechanics as well
as Artificial Intelligence and Machine learning for our web-app, which I will discuss in the next point.

The web-app

The web-app, which you can find here is a game with an AI. We didn't program the game fully by ourselfes, but we know very good how it works and what it does.
The game is a copy of the populuar mobile game HillClimbRacing. The player is in a 2D Landscape and sits in the car, and only with braking and stepping on the gas you have to
get as far as possible. The landscape goes onto infinity.
Because this isn't really what a self-driving cars defines, we figured we would just put an A.I. on it.
The principe we used is called N.E.A.T. NEAT stands for NeuroEvolution of Augmenting Topologies. The thing about NEAT is, it goes after natural selection.
That means that if I had 100 cars in one Generation, the 50 worst cars get killed and new cars are being created that learn from their ancestors mistakes.
That's exactly what our game does. Per Generation we have 50 "players" which are essentially just different AIs. Each of them just trys to brake or to step on the gas.

After each generation the A.I. gets better and better. You can always try it yourself by pressing "P", Fight against the AI by pressing "F" after the first generation, or press "G" to see the
the best A.I. generated so far.
Of course you can also stop the game by pressing "P".
